# Installation

pyXLMS is a [python](https://www.python.org/) package, naturally it requires a [python installation](https://www.python.org/downloads/).
pyXLMS supports python version 3.7 and greater!

## Installation via PyPI

pyXLMS is available on [PyPI](https://pypi.org/project/pyXLMS/) and can easily be installed via pip:
```
pip install pyxlms
```

## Installation via GitHub

Alternatively, you can install the latest stable version directly from [GitHub](https://github.com/hgb-bin-proteomics/pyXLMS):
```
pip install git+https://github.com/hgb-bin-proteomics/pyXLMS.git
```

_or the latest development version:_
```
pip install git+https://github.com/hgb-bin-proteomics/pyXLMS.git@develop
```
