# Introduction

pyXLMS supports the following crosslink search engines and input file formats for
reading crosslink results at both crosslink-spectrum-match and crosslink/residue pair
level.

## Available Parsers with Documented Tutorial Pages

- [Reading pyXLMS and Custom Files](/docs/parsers/custom)
- [Reading MaxLynx/MaxQuant Result Files](/docs/parsers/maxlynx)
- [Reading MeroX Result Files](/docs/parsers/merox)
- [Reading MS Annika Result Files](/docs/parsers/msannika)
- [Reading mzIdentML Result Files](/docs/parsers/mzid)
- [Reading pLink Result Files](/docs/parsers/plink)
- [Reading Scout Result Files](/docs/parsers/scout)
- [Reading xiSearch/xiFDR Result Files](/docs/parsers/xi)
- [Reading XlinkX Result Files](/docs/parsers/xlinkx)

## Overview of Supported Search Engines and Formats

The following abbreviations are used:
- **CSMs:** crosslink-spectrum-matches
- **XLs:** crosslinks/residue pairs

The following crosslink search engines and formats are currently supported:

| Search Engine/Format | Parser            | CSMs | XLs | Supported Formats            | Page                           | API Docs |
|:---------------------|:------------------|:----:|:---:|:-----------------------------|:-------------------------------|:---------|
| MaxQuant/MaxLynx     | `read_maxquant()` | ✅  | ❌  | `.txt`                       | [link](/docs/parsers/maxlynx)  | [link](https://hgb-bin-proteomics.github.io/pyXLMS/pyXLMS.parser.html#pyXLMS.parser.parser_xldbse_maxquant.read_maxquant) |
| MeroX                | `read_merox()`    | ✅  | ❌  | `.csv`, `.zhrm`              | [link](/docs/parsers/merox)    | [link](https://hgb-bin-proteomics.github.io/pyXLMS/pyXLMS.parser.html#pyXLMS.parser.parser_xldbse_merox.read_merox) |
| MS Annika            | `read_msannika()` | ✅  | ✅  | `.pdResult`, `.xlsx`, `.txt` | [link](/docs/parsers/msannika) | [link](https://hgb-bin-proteomics.github.io/pyXLMS/pyXLMS.parser.html#pyXLMS.parser.parser_xldbse_msannika.read_msannika) |
| pLink 2 & 3          | `read_plink()`    | ✅  | ✅  | `.csv`                       | [link](/docs/parsers/plink)    | [link](https://hgb-bin-proteomics.github.io/pyXLMS/pyXLMS.parser.html#pyXLMS.parser.parser_xldbse_plink.read_plink) |
| Scout                | `read_scout()`    | ✅  | ✅  | `.csv`                       | [link](/docs/parsers/scout)    | [link](https://hgb-bin-proteomics.github.io/pyXLMS/pyXLMS.parser.html#pyXLMS.parser.parser_xldbse_scout.read_scout) |
| xiSearch & xiFDR     | `read_xi()`       | ✅  | ✅  | `.csv`                       | [link](/docs/parsers/xi)       | [link](https://hgb-bin-proteomics.github.io/pyXLMS/pyXLMS.parser.html#pyXLMS.parser.parser_xldbse_xi.read_xi) |
| XlinkX               | `read_xlinkx()`   | ✅  | ✅  | `.pdResult`, `.xlsx`, `.txt` | [link](/docs/parsers/xlinkx)   | [link](https://hgb-bin-proteomics.github.io/pyXLMS/pyXLMS.parser.html#pyXLMS.parser.parser_xldbse_xlinkx.read_xlinkx) |
| mzIdentML            | `read_mzid()`     | ✅  | ❌  | `.mzid`                      | [link](/docs/parsers/mzid)     | [link](https://hgb-bin-proteomics.github.io/pyXLMS/pyXLMS.parser.html#pyXLMS.parser.parser_xldbse_mzid.read_mzid) |
| pyXLMS/custom        | `read_custom()`   | ✅  | ✅  | `.xlsx`, `.parquet`, `.csv`  | [link](/docs/parsers/custom)   | [link](https://hgb-bin-proteomics.github.io/pyXLMS/pyXLMS.parser.html#pyXLMS.parser.parser_xldbse_custom.read_custom) |

> [!TIP]
>
> **➡️ All search engine parsers also support any text-like file format that uses a character delimiter e.g. `.csv`, `.tsv` and tab-delimited `.txt`.**
