# Introduction

## Available Figures and Plot Types

pyXLMS currently features several different ways to plot your data:

- [Plotting a crosslink type distribution](/docs/plotting/plot_crosslink_type_distribution)
- [Plotting a peptide pair distribution](/docs/plotting/plot_peptide_pair_distribution)
- [Plotting a protein distribution](/docs/plotting/plot_protein_distribution)
- [Plotting a crosslink-spectrum-match or crosslink score distribution](/docs/plotting/plot_score_distribution)
- [Plotting a target and decoy distribution](/docs/plotting/plot_target_decoy_distribution)
- [Plotting Venn diagrams](/docs/plotting/plot_venn_diagram)
